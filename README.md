This repository contains materials for a statistics coursework project. The project simulates data based on a study examining sleep-related neural activation and threat perception. The analysis reproduces a Generalised Linear Mixed Model (GLMM) examining the effect of duration-related neural activation on threat-related decisions while controlling for face valence.

In addition, power and sensitivity analyses were conducted to evaluate how changes in sample size and effect size influence the detectability of the target effect.
